# 2026-06-26 Lessons Learned

- Adapter work should wait until the shared core is stable.
- `NullPool` should be described honestly; it is not a full connection pool.
- Verification should prove the formatter directly, not only through database queries.
- Documentation and Verification need different audiences and different file styles.
- A Cockpit should separate reading documents from raw evidence so new members do not get lost.
