本目录按历史需求项目归档旧版本报告、过程记录和验证记录。

| 项目目录 | 内容 |
|---|---|
| `01_Dify本地化插件制作流程/` | Dify 本地化插件制作流程的历史 qmd/html 版本 |

| `Phase5_5_Workflow_Automation_Completion/` | Workflow 自动化完成阶段报告 |

归档规则：同一项目阶段放在同一子目录；`最新报告/` 只保留当前阶段报告；每次生成新报告时，上一份报告整体移入对应 Phase 目录。
