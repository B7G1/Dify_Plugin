# v1.0 Public Demo

This directory contains a five-minute, evidence-first demonstration of the frozen baseline. It does not create data, change Provider credentials, or expose API keys.

## Audience takeaway

By the end, the audience should understand that one Dify Tool provides a safe, extensible database-query path and that the claim is backed by a real Workflow, API execution, cold boot, and 45 automated checks.

## Files

- `demo_script.md`: exact spoken script and screen actions.
- `demo_checklist.md`: preflight, privacy, fallback, and teardown.
- `obs_scene_plan.md`: OBS scenes and source ordering.
- `powerpoint_sequence.md`: supporting slide order for the demo.

Use the live system for execution. Use screenshots only as a fallback, and only those marked public-ready in `docs/images/README.md`.
