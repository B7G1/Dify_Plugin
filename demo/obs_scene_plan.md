# OBS Scene Plan

Use a 1920×1080 canvas and a single display-capture source. No crop, mask, blur, overlay, or post-production redaction is allowed for evidence scenes.

| Scene | Source | Purpose |
| --- | --- | --- |
| 01 Opening | PowerPoint overall architecture | explain the end-to-end contract |
| 02 Plugin | fullscreen Dify Plugin page | show installed 0.1.1 |
| 03 Provider | Plugin page with success toast only | prove credential validation without fields |
| 04 Workflow | fullscreen Workflow | show Start → Tool → Output |
| 05 Result | Workflow Result tab | show `SELECT 1`, DM, success, JSON |
| 06 Verification | clean PowerShell | show 45/0/0 |
| 07 Dashboard | fullscreen local Dashboard | close on maintainability and evidence |

Disable OBS preview thumbnails if they could retain sensitive prior frames. Use hard cuts, not animated transitions. Keep a “Safe Hold” scene containing only the overall architecture for unexpected UI states.
