# Presentation Demo Flow

Use the five-minute demo between slides 6 and 9.

1. Slide 6 transitions to fullscreen Dify Plugin page.
2. Show installed 0.1.1 without opening credentials.
3. Show Provider validation success toast with the form closed.
4. Run Workflow `SELECT 1`; show Result, not Details.
5. Run Workflow verifier or show safe API result.
6. Show real 45/0/0 terminal evidence.
7. Return to slide 9 and interpret the evidence.

Fallback: if the live environment fails, switch to the Safe Hold architecture slide and use only screenshots marked public-ready. Never debug authentication or open credential settings during a presentation.
